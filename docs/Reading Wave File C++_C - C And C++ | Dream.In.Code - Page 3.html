Re: Reading wave file C++/C

Posted 20 November 2013 - 09:01 AM

Your lines 144-151 are wrong. The lines prior to that read in two bytes or one byte depending on whether its 16 bit samples or not. If it's 16 bit samples, you need to assemble the two bytes to form an twos-complement integer. If it's an 8 bit sample you just take the single byte as is. Then you need to scale appropriately depending on 16 or 8 bit.

You are not doing the combining of the two bytes. You are simply scaling the two bytes and outputting two values for every sample value.
